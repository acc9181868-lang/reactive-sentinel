# Reactive Sentinel MVP Roadmap

## Goal
Build a public beta that tracks whale movements, exchange inflows/outflows, burns, and staking events.

## Week 1
- Landing page
- Alert feed UI
- Placeholder dashboard
- Define tracked wallet list

## Week 2
- Backend event watcher
- RPC connection
- Telegram bot alerts
- Basic confidence score

## Week 3
- Database storage
- Real dashboard data
- User-created alerts

## Week 4
- Reactive Network integration
- Public beta
- Community feedback
